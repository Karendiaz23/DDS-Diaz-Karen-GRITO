# 🚀 Manual de ejecución – Lista de Compras

Repositorio del proyecto:
https://github.com/Karendiaz23/DDS-Diaz-Karen-GRITO.git

---

## 1. Clonar el repositorio

```bash
git clone https://github.com/Karendiaz23/DDS-Diaz-Karen-GRITO.git
```

---

## 2. Entrar al proyecto

```bash
cd DDS-Diaz-Karen-GRITO
```

---

## 3. Instalar dependencias

```bash
bun install
```

---

## 4. Iniciar la aplicación

```bash
bun start
```

---

## 5. Ejecutar la app

* Descargar Expo Go en el celular
* Escanear el código QR que aparece en la terminal o navegador

---

## Resultado

La aplicación se ejecutará correctamente y permitirá gestionar la lista de compras.
